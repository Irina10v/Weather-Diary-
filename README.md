Описание программы
Weather Diary - это GUI-приложение для ведения дневника погоды. Приложение позволяет добавлять записи о погоде, фильтровать их по дате и температуре, сохранять и загружать данные в формате JSON, а также удалять записи.

Возможности:
Добавление записей (дата, температура, описание, осадки)
Фильтрация по дате (точное совпадение)
Фильтрация по температуре (выше/ниже/равно)
Сохранение данных в JSON-файл
Загрузка данных из JSON-файл
Удаление выбранной записи
Очистка всего списка
Валидация вводимых данных (проверка пустых полей, формат даты, тип температуры)
Инструкция по запуску
Требования:
Python 3.6 или выше
Tkinter (входит в состав Python)
Запуск:
Откройте терминал или командную строку
Перейдите в директорию проекта:
cd weather-diary
Запустите приложение:
python weather_diary.py
Примеры использования
Добавление записи:
Заполните все поля формы:
Дата: "05.05.2026"
Температура: "25"
Описание: "Солнечно"
Осадки: "Нет"
Нажмите кнопку "ДОБАВИТЬ"
Запись появится в списке справа
Фильтрация:
По дате: Введите дату в поле "ФИЛЬТР ПО ДАТЕ" и нажмите "НАЙТИ ПО ДАТЕ"
По температуре: Введите значение и нажмите "ВЫШЕ", "НИЖЕ" или "РАВНО"
Показать все: Нажмите "ПОКАЗАТЬ ВСЕ"
Сохранение и загрузка:
Нажмите "СОХРАНИТЬ В JSON" - задачи сохранятся в файл weather_data.json
Нажмите "ЗАГРУЗИТЬ ИЗ JSON" - задачи загрузятся из файла weather_data.json
Удаление:
Выберите запись в списке и нажмите "УДАЛИТЬ"
Для удаления всех записей нажмите "ОЧИСТИТЬ ВСЁ"
Тестовые данные (примеры для проверки):
Попробуйте добавить следующие записи для тестирования:

Дата	Температура	Описание	Осадки
01.05.2026	18	Солнечно	Нет
02.05.2026	12	Пасмурно	Нет
03.05.2026	8	Дождь	Да
04.05.2026	-2	Снег	Да
05.05.2026	22	Тепло	Нет
После добавления попробуйте:

Отфильтровать записи с температурой выше +15°C
Найти записи с осадками (да)
Найти записи с датой "05.05.2026"
Найти записи с температурой равной 15°C
Сохранить данные в JSON, закрыть приложение, затем загрузить обратно
Удалить несколько записей и очистить список
Валидация данных
Все поля должны быть заполнены
Формат даты должен быть ДД.ММ.ГГГГ (например, 05.05.2026)
Температура должна быть числом (целым или дробным)
При удалении необходимо выбрать запись из списка
При очистке всего списка подтверждение не требуется (по желанию можно добавить)
